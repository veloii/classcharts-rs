## What is this?

An API wrapper with *mostly* proper typings and tests for ClassCharts.

## Using as a library
```bash
cargo add classcharts
```
or in your `Cargo.toml`
```toml
[dependencies]
...
classcharts = "1.0.0"
```

## Developing

```bash
git clone git@github.com:veloii/classcharts-rs.git
cd classcharts-rs
```
```bash
cargo test
```
